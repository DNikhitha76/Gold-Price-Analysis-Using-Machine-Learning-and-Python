Step-1: Create a Project Folder
put all files in the same folder:
Goldpriceproject
1.1 gold_price_data.csv
1.2 gold_analysis_backend.py
1.3 streamlit_app.py

Step-2: Install Required Libraries
Open Command Prompt or VS Code Terminal.
pip install streamlit pandas numpu matplotlib seaborn plotly scikt-learn

Step-3: Open Project Folder
cd path_to_your_project_folder
Example:
cd D: \Projects\Goldpriceproject

Step-4: Run Streamlit App
Execute:
streamlit run streamlit_app.py
If your file name contains (1):
streamlit run "streamlit_app(1).py"
